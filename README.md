# InventoryManager

A module to track inventory levels and generate alerts for low stock.

## Goal
Track inventory levels and generate alerts for low stock.

## Acceptance Criteria
1. `inventory` module exists and can be imported.
2. `add_item` function adds an item to inventory.
3. `get_stock` function returns quantity for a given item.
4. `check_low_stock` function returns items with quantity below a threshold.
